Corde's configuration can be defined through a `corde.config.js|json|ts` file. This file must be at the root of your application,
or you can declare a custom path for it with the cli option [--config](CLI.mdx#--config).

This is the default config file for Corde:

```javascript
{
    "cordeBotToken":  "YOUR_TESTING_BOT_TOKEN_HERE",
    "botTestId":  "YOUR_TESTING_BOT_ID_HERE",
    "botToken":  "YOUR_BOT_TOKEN_HERE",
    "guildId":  "THE_GUID_OF_BOT_HERE",
    "channelId":  "CHANNELS_ID_HERE",
    "botPrefix":  "+",
    "testMatches":  ["./test/**"],
    "timeout": 1000
}
```

:::note
both `cordeBotToken` and `botToken` can be found in [Discord Developers portal](https://discord.com/developers/applications)
:::

## Configuration Options

### `cordeBotToken`

The bot token that Corde will use to simulate a user.

**Required: `true`**  
**Type: `string`**  
**Example: `YaA4MDMiOTY2O4I2MjAwODMy.X2iRwg.Rf3-TqLExWuPQjxnVaDCGv9V7cB`**

---

### `botTestId`

Your bot's id.

**Required: `true`**
**Type: `string`**  
**Example: `514212632960122287894`**

---

### `botToken`

Your bot's Token.

**Required: `true`**  
**Type: `string`**  
**Example: `YaA4MDMiOTY2O4I2MjAwODMy.X2iRwg.Rf3-TqLExWuPQjxnVaDCGv9V7cB`**

---

### `guildId`

The id of the `guild` where both bots are.

**Required: `true`**  
**Type: `string`**  
**Example: `514212632960122287894`**

---

### `channelId`

The id of the `channel` where both bots are.

**Required: `true`**  
**Type: `string`**  
**Example: `514212632960122287894`**

---

### `botPrefix`

Prefix for call your bot.

**Required: `true`**  
**Type: `string`**  
**Example: `!`**

---

### `testMatches`

Array with the path of test folders | files.

**Required: `true`**  
**Type: `string`**  
**Example: `['./tests']`**

---

### `timeout`

Timeout for each test. The value in milliseconds.

**Required: `false`**  
**Type: `number`**  
**Example: `1000`**  
**Default: `5000`**

---

### `modulePathIgnorePatterns`

Define the file pattern that corde should ignore.

**Required: `false`**  
**Type: `string[]`**  
**Example: `['**/tests/\*.fixture.js']`\*\*

---

### `rootDir`

Defines root dir of the project.

**Required: `false`**  
**Type: `string[]`**  
**Default: `process.cwd()`**

---

### `extensions`

**Define file extensions to be loaded**

**Required: `false`**  
**Type: `string[]`**  
**Default: `[".js",".ts"]`**

---

### `exitOnFileReadingError`

Define if corde should stop if any problem occour when importing a test file.

**Required: `false`**  
**Type: `boolean`**  
**Default: `true`**

---

### `project`

Definition of tsconfig path.

**Required: `false`**  
**Type: `string`**  
**Default: `<rootDir>/tsconfig.json`**
